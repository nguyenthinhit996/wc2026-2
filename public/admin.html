Bảng điều khiển

Nhập kết quả trận, thêm cặp đấu, import lịch từ file JSON. Mỗi thao tác ghi đều tự động sao lưu.

Sao lưu dữ liệu

Tải toàn bộ người chơi, trận đấu và dự đoán về máy. Nên làm trước khi import file lớn.

Import lịch / kết quả (JSON)

File là mảng các trận. Hệ thống kiểm tra định dạng, cho xem trước, rồi gộp theo match_id. Xem mẫu định dạng

[
  {
    "match_id": "m010",
    "round": "Tứ kết",
    "team1": "Pháp",
    "team2": "Anh",
    "kickoff_time": "2026-07-05T02:00:00+07:00",
    "status": "upcoming",
    "actual_result": null,
    "actual_score": null
  }
]

Thêm / sửa cặp đấu

Nhập match_id trùng để cập nhật trận có sẵn. Tuần được tính tự động từ giờ thi đấu.
